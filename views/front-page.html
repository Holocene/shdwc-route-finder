Summary

This is a tool for keeping track of wormholes, and also route finding through them. It is tested in Firefox and Chrome.

Check out the 'route finder' tab, which is the most interesting thing in the system right now.

If you have any suggestions or notice any bugs, post them to the Shadow Cartel forums, add an issue in the issue tracker or contact me in game: Olixia Castitatis.

The site is developed using javascript, html and css. Angular JS provides the framework in which the site is built. The site is hosted by Github, and the database is provided by Firebase. Some backend services are provided by Microsoft Azure.

How do I use this tool?

Video Guide

Check out this 12 minute guide to probing wormholes and using the tool: https://youtu.be/eq8OnXZKwp8

Scanning

  1. Go to the systems page
  2. Find the system you are in.
  3. In EVE, scan some signatures with your probe scanner. Select them all, then copy them to clipboard (hint: ctrl-A, ctrl-C).
  4. Click 'paste scan results' to copy your results into the tool
  5. Add any wormholes you find.

If you add the signatures you find to the tool, it's easier for the next prober to work out what sigs are new and what have stayed the same.

Route finding

Go to the route finding page. Enter your start and destination system and you'll see an array of routes. Use the filters on the right to set system restrictions (no highsec) or transport method restrictions (no titans, or no wormholes).
